# 媒体文件管理器

一个功能强大的媒体文件管理工具，支持图片和视频文件的扫描、预览、搜索和收藏。

## 功能特性

- **文件扫描**：支持扫描指定目录下的图片和视频文件
- **多格式支持**：支持多种图片和视频格式
- **实时预览**：扫描过程中实时显示文件预览
- **文件搜索**：支持按关键词搜索文件
- **文件筛选**：支持按文件类型（全部、仅图片、仅视频）筛选
- **文件排序**：支持按名称、大小、修改时间排序
- **文件收藏**：支持将文件添加到收藏夹
- **定位文件**：支持打开文件所在的文件夹
- **历史记录**：保存扫描历史记录
- **网格/列表视图**：支持两种查看模式
- **图片缩放**：支持Ctrl+滚轮调整图片大小

## 安装方法

1. **安装依赖**
   ```bash
   pip install -r requirements.txt
   ```

2. **运行程序**
   ```bash
   python main.py
   ```

## 使用方法

### 扫描文件
1. 在顶部的路径输入框中输入要扫描的目录路径
2. 选择文件类型筛选（全部、仅图片、仅视频）
3. 点击"开始扫描"按钮
4. 等待扫描完成，扫描结果会显示在左侧展示区

### 查看文件
1. 在左侧展示区中点击文件查看预览
2. 可以在网格视图和列表视图之间切换
3. 使用Ctrl+滚轮调整图片大小
4. 点击右侧预览区的"定位到文件夹"按钮可以打开文件所在的文件夹

### 搜索文件
1. 在顶部的搜索框中输入关键词
2. 搜索结果会实时显示

### 收藏文件
1. 在预览区点击"添加到收藏夹"按钮
2. 选择或创建收藏夹
3. 可以在"收藏夹"菜单中查看收藏的文件

## 注意事项

1. **文件名称不要过长**：文件名称过长可能会导致无法正常工作，建议保持文件名称在合理长度内
2. **路径长度限制**：Windows系统对路径长度有限制，建议扫描的目录路径不要过长
3. **文件格式支持**：请确保文件格式在支持列表中
4. **性能优化**：扫描大量文件时可能会消耗较多资源，请耐心等待
5. **权限问题**：确保程序有足够的权限访问扫描的目录

## 支持的文件格式

### 图片格式
- .jpg, .jpeg, .png, .gif, .bmp, .tiff, .webp, .raw, .heic, .psd, .ai

### 视频格式
- .mp4, .mkv, .mov, .avi, .flv, .wmv, .rmvb, .webm, .m4v, .ts

## 技术栈

- **Python**：主要开发语言
- **PyQt6**：GUI框架
- **SQLite**：数据库存储
- **OpenCV**：视频处理
- **Pillow**：图像处理
- **FFmpeg**：视频解码
