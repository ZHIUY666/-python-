# 媒体文件管理器 支持图片和视频文件的扫描、预览、搜索和收藏。  ## 功能特性  - **文件扫描**：支持扫描指定目录下的图片和视频文件 - **多格式支持**：支持多种图片和视频格式 - **实时预览**：扫描过程中实时显示文件预览 - **文件搜索**：支持按关键词搜索文件 - **文件筛选**：支持按文件类型（全部、仅图片、仅视频）筛选 - **文件排序**：支持按名称、大小、修改时间排序 - **文件收藏**：支持将文件添加到收藏夹 - **定位文件**：支持打开文件所在的文件夹 - **历史记录**：保存扫描历史记录 - **网格/列表视图**：支持两种查看模式 - **图片缩放**：支持Ctrl+滚轮调整图片大小
